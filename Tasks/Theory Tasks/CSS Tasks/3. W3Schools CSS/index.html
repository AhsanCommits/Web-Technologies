CSS properties W3Schools

CSS Comments

CSS comments are used to add notes or descriptions to your CSS code. They are not displayed on the webpage but can be helpful for other developers who might work on the same code. Comments start with `/*` and end with `*/`. 

                ```css
                /* This is a CSS comment */

CSS Colors

CSS colors are used to set the color of text, background, borders, and other elements on a webpage. Colors can be specified in different formats such as color names, RGB values, HEX values, HSL values, and more. 


                body {
                color: red;
                background-color: yellow;
                }

CSS Backgrounds

CSS backgrounds are used to set the background image or color of an element. You can use different properties to set the background, such as background-color, background-image, background-repeat, background-position, and more. 

            body {
                background-color: gray;
                background-image: url("background.jpg");
                background-repeat: no-repeat;
                background-position: center;
            }

CSS Borders

CSS borders are used to create a border around an element. You can use different properties to set the border, such as border-style, border-width, border-color, and more. 

                    div {
                    border-style: solid;
                    border-width: 2px;
                    border-color: red;
                    }

CSS Margins

CSS margins are used to create space around an element. You can use different properties to set the margin, such as margin-top, margin-right, margin-bottom, and margin-left. 

                    div {
                    margin: 10px;
                    }

CSS Padding

CSS padding is used to create space within an element. You can use different properties to set the padding, such as padding-top, padding-right, padding-bottom, and padding-left. 

                    div {
                    padding: 10px;
                    }

CSS Height/Width

CSS height and width are used to set the height and width of an element. You can use different units to specify the height and width, such as pixels, percentages, ems, and more. 

                    img {
                    height: 100px;
                    width: 100px;
                    }

CSS Box Model

CSS box model is used to calculate the size of an element. It consists of the content area, padding area, border area, and margin area. 

                    div {
                    width: 100px;
                    height: 100px;
                    padding: 10px;
                    border: 2px solid red;
                    margin: 10px;
                    }

CSS Outline

CSS outline is used to create an outline around an element. It is similar to border but it does not take up space and it is drawn outside the border. 

                    div {
                    outline: 2px solid blue;
                    }

CSS Text

CSS text is used to set the style of text on a webpage. You can use different properties to set the font, size, color, alignment, and more. 

                    p {
                    font-family: Arial, sans-serif;
                    font-size: 16px;
                    color: black;
                    text-align: center;
                    }

CSS Fonts

CSS fonts are used to set the font family and font size of text on a webpage. You can use different font families and sizes to achieve the desired look. 

                    p {
                    font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;
                    font-size: 18px;
                    }
